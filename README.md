📦 LocalBox Miner – React CRUD App with Local Storanodege
📌 Project Overview

LocalBox Miner is a simple React.js CRUD application that allows users to store and manage records using browser Local Storage. The application provides a clean interface where users can add, edit, and delete user information such as Name and Email. All data is stored locally in the browser, so it persists even after refreshing the page.

This project demonstrates core React concepts such as state management, event handling, component rendering, and local storage integration.

✨ Features

➕ Add new user records (Name & Email)

📋 Display stored records in a table

✏️ Edit existing records

❌ Delete records

💾 Data persistence using Local Storage

🔢 Display total number of records

🎨 Clean and responsive UI

🖥️ Application Interface
1️⃣ Empty State (No Records)

When the application starts and no data is stored in Local Storage, it displays:

Input fields for Name

Input fields for Email

Add Button

Message: "No Records Found"

2️⃣ Records Stored

Once the user adds data:

The record appears in a table format

Each record contains:

Name

Email

Edit Button

Delete Button

Total records count is displayed at the bottom.

⚙️ Technologies Used

⚛️ React JS

🧠 React Hooks (useState, useEffect)

💾 Local Storage API
